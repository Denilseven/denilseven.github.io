O Pingo do Conhecimento

The goal of this project is to help kids learn through video-games!
The main point here is the game Pingo no Mundo da Matemática (Pingo in the World of Mathematics), a platformer about an adventurous cat set on a quest to free his world from math monsters that are causing trouble.

My vision is for it to be played in schools as an uplifting activity that can help students learn and practice math while having fun - and with technologies like Chromebooks being more available in schools near where I live, I can totally see that happening!

This project started in September 2023 with the support of my English teacher from school. I developed an alpha version of the game to be presented at a state science and innovation fair, and it was a total success! The project got founded, and by the end of this year I'll be releasing a complete version of the game to be played in schools and by anyone without any costs.

You can play the game's apha version at the project's website!
